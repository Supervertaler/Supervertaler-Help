## What each product does

<CardGrid stagger>
  <Card title="🧩 Supervertaler for Trados" icon="puzzle">
    A plugin for [Trados Studio](https://www.trados.com/) that adds
    AI-powered translation, terminology, and a hands-free assistant
    to the CAT tool you already use.

    Source-available, paid (three tiers).

    [Open the Trados Plugin docs →](/trados/)
  </Card>

  <Card title="🖥️ Supervertaler Workbench" icon="laptop">
    A standalone, free, open-source desktop translation environment
    with its own editor, AI integration, terminology, translation memory,
    and a system-wide companion window (Sidekick) for voice commands and
    quick lookups.

    [Open the Workbench docs →](/workbench/)
  </Card>
</CardGrid>

## Not sure which one you need?

- If you already pay for **Trados Studio** and want to add AI translation, terminology assistance, and voice commands to it → **Supervertaler for Trados**.
- If you want a **free, modern translation tool** that doesn't depend on any other CAT software → **Supervertaler Workbench**.
- If you do **both**, you can install both — they share AI providers and a coherent workflow.
